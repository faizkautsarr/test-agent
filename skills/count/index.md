# Count Skill

Count the number of characters in the input string and return the length.

## Usage

```
/test-agent:count "<string>"
```

## Examples

- `/test-agent:count "aaa"` → 3
- `/test-agent:count "hello world"` → 11

## Instructions

The user has provided the following input: $ARGUMENTS

Strip any surrounding quotes from the input, then count the number of characters in the string and respond with only the number.
